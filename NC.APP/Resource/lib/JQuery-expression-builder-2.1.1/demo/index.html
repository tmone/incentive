Expression Builder Demo

Example 1

Simple use of the tool with the following json data: 

let options = {
      variables: [
        {
          id: 1,
          name: 'Age'
        },
        {
          id: 2,
          name: 'Gender'
        }
      ]
};
expressionBuilder('#txt', options);

To read the output here are some examples: 

let exp = expressionBuilder('#txt').getExpression();

Example 2

In the following example the expression has been set by default: 

let options = {
      variables: [
        {
          id: 1,
          name: 'FirstName'
        },
        {
          id: 2,
          name: 'LastName'
        }
      ],
      suggestions: 'up',
      expression: '[2] + 35'
};
expressionBuilder('#txt2', options);

To set the expression here is an example: 

expressionBuilder('#txt2').setExpression('([1] + 15) + 1');
